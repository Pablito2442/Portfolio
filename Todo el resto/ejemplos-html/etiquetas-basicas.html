Los títulos

Las etiquetas h1,h2,...,h6 se utilizan para establecer títulos.

Obviamente, no utilizaremos un h2 si no hemos utilizado h1 previamente.

Los párrafos

Mediante la etiqueta p creamos párrafos en los documentos.

Si queremos crear un salto de línea
utilizamos la etiqueta br.

Las listas

Podemos definir listas ordenadas y listas desordenadas.

Las listas ordenadas se crean mediante la etiqueta ol y dentro de ésta cada item mediante la etiqueta li:

  1. Elemento 1.
  2. Elemento 2.

Las listas desordenadas se crean exactamente igual pero utilizando la etiqueta ul:

  • Un elemento.
  • Otro elemento.
  • Otro más.

Las tablas

Para definir tablas utilizamos la etiqueta table:

  1. Creamos dentro una etiqueta tr para cada fila.
  2. Dentro de cada tr, crearemos una etiqueta td para definir cada celda.
  3. Si la celda es un encabezados, la definiremos con th y no con td.
Columna 1 Columna 2
Fila 1, Columna 1 Fila 1, Columna 2
Fila 2, Columna 1 Fila 2, Columna 2

Si queremos crear tablas más complejas, es posible que tengamos que agrupar columnas o filas:

  • Para agrupar columnas usamos el atributo colspan
  • Para agrupar filas usamos el atributo rowspan

Los enlaces

Los enlaces son una parte fundamental de la Web ya que mediante éstos navegamos entre las páginas de un sitio.

Se declaran mediante la etiqueta a, indicando en el atributo href dónde está el fichero de destino. El contenido de la etiqueta, indicará el texto que se le mostrará al usuario.

Podemos crear enlaces a URLs externas (URL con HTTP): IR A GOOGLE. Fíjate que este texto sigue a continuación del enlace¡La etiqueta a no es una etiqueta de bloque!

Si queremos que el enlace se abra en una nueva pestaña, usamos el atributo target con el valor _blank : IR A GOOGLE.

Al igual que con las imágenes, podemos hacer referencia a otros documentos (HTML o no) que tengamos en el servidor: Página auxiliar o Mostrar Zarigüeya.
